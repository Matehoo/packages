## Usage

Install package:

```sh
# npm
npm install @kodadot1/static

# yarn
yarn add @kodadot1/static

# pnpm
pnpm install @kodadot1/static
```

Import:

```js
// ESM
import {} from "@kodadot1/static";

// CommonJS
const {} = require("@kodadot1/static");
```
